## 변수 작성법

## 변수 작성 시 주의사항
* 변수의 이름은 문자(a~z, A~Z), 숫자(0~9), 달러($), 밑줄(_)이 포함된다.
* 변수의 이름은 숫자로 시작할 수 없다.
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수없다.
* 키워드나 예약어는 사용할 수없다.
    * 키워드 예약어 종류 -> `delete, this, false, ture, if, default, var, new, class`등
* 모든 변수는 대/소문자를 구별합니다. `score`와 `Score`는 다른 변수!
* ★ 변수가 저장할 정보의 종류를 표현하는 의미 있는 단어를 사용해야함!
## 변수 표기법 종류

## null과 undefind 차이
### null
* 값이 비어있음, 존재하지않음
* 사용자에 의해 결정되는 의도적인 대입 값
### undefind
* 정의(할당)되지않음
* undefind는 변수를 초기화하지않은 오류상태
* undefind를 변수에 대입해선 안된다
* 사용자가 실수로 초기화하지않은 변수에 들어간 일종의 오류값

## 산술연산자 -> 피연산자가 숫자인경우
* `a+b` 더하기
* `a-b` 빼기
* `a*b` **★곱하기**
* `a/b` **★나누기**
* `a%b` **★나머지** : 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지이다
* `a**b` 거듭제곱 : 왼쪽의 피연산자의 오른쪽 피연산자의 값만큼 곱한 값이다.
* 계산할 수 없는 경우는 NaN 처리 -> ex)'two'*2=NaN
### 연결연산자(+), 더하기연산자(+)
* `문자열 + 문자열` 예)`'hello' + 'world'` : 문자열 결합으로 더하기 연산이 아닌 연결 연산자로 인식
* `문자열 + 숫자` 예)`'hello' + 7` : 문자열 결합으로 더하기 연산이 아닌 연결 연산자로 인식
* `숫자 + 숫자` 예)`1 + 7` : 숫자 결합으로 더하기 연산자로 인식
* `(숫자 + 숫자) + 문자열` 예)`(1+7) + 'world'` : 앞 순자는 더하기로 처리후 최종적으로 문자열로 결합
### 빼기(-) 곱하기(*) 나누기(/) 연산자의 특별한 기능
* 더하기 연산자는 피연산자에 따라 문자열이 하나라도 있으면 문자열 결합으로 결과가 나타나지만 빼기, 곱하기, 나누기 연산자는 **숫자형으로 자동 형변환하여 인식**
* `문자열 * 숫자` 예) `'1'* 7` : 문자를 숫자로 자동 형변환하여 결과값을 처리
* `숫자 / 숫자` 예) `1 / 7` : 숫자로 인식 후 결과값을 처리
* `(숫자 + 숫자) - 문자열` 예) `(1 + 7) * '2'` : 문자를 숫자로 자동 형변환하여 결과값을 처리

### 복합대입연산자
* 코드의 간결함을 위해 사용, 사칙연산 모두 복합대인 연산자 적용이 가능
* **기존**데이터가 변경 될 때 사용된다.

* `-=`  예) `a -= b` -> `a = a - b` :  기존 a값에 b를 빼서 a에 대입
* `*=`  예) `a *= b` -> `a = a * b` :  기존 a값에 b를 곱해서 a에 대입
* `/=`  예) `a /= b` -> `a = a / b` :  기존 a값에 b를 나눠서 a에 대입
* `%=`  예) `a %= b` -> `a = a % b` :  기존 a값에 b를 나눈 나머지값을 a에 대입

## 사용자정의함수 작성법
* `function` : 함수선언 시 함수라는 것을 나타내는 키워드
* `함수이름` : 개발자가 만든 함수의 의미있는 이름을 작성한다
* `전달인자` : 매개변수, 인수 등 여러가지 용어를 사용한다. 함수내에 데이터를 전달하는 변수명이라 보면 되며 **전달 데이터**가 많은 때 객체 타입으로 전달하기도 한다.(필수아님)(예:조약돌)
* `중괄호{}` : 함수 영역의 시작과 끝을 나타내며 함수의 반복적인 실행 내용을 작성
* `fonction 함수명(전달인자){함수실행내용}`
### 콜백함수
* 사용자정의함수 또는 내장함수를 특정조건에 따라 실행하는 함수.
    * 함수의 종류가 아닌 `사용방식(역할)`에 가까운 목적
    1. 내가 직접 운전(실행)하는 것이 아닌, 대리기사님(다른 함수)에게 운전을 요청
    2. 사용자정의함수 생성 : 집에 도착하면(조건) 깨워주세요(콜백실행)
    3. 기사님(다른함수)은 운전을 하다가 '집도착'이라는 조건이 충족되면 나를 깨우는 행동(콜백)을 실행

## 매개변수
* 함수가 반복될 때마다 달라질 수있는 데이터
* 전달하는 인자 : 외부에서 함수로 특정데이터를 전달하는 방법
* 매개변수는 지역변수로 처리 됨
* 매개변수를 잘 활용할 줄 알아야 함수의 효율성이 향상됨
* 매개변수 생성과 동시에 기본 값을 설정할 수있음 **매개변수=기본값**
* 매개변수 갯수 제한은 없지만, 아예 없거나 보통 1~2개 정도가 사용된다.
### 매개변수 작성법

## 배열(Array)
* 여러 개의 값(데이터)을 순서대로 저장할 수있는 자료구조.
* `[대괄호]`를 사용해서 각 데이터를 저장
* 각 개별은 인덱스(index)로 접근하며 **숫자는 0부터 인식(인덱스번호)**
* 데이터 전체 **길이(length)는 1부터 인식**
### 배열 생성과 호출
* let 변수명 =[배열, 배열, 배열···]
* let 변수명 = new Array(배열, 배열, 배열···)
* let 변수명 = new Array(3)
* (위) 배열의 다양한 생성방법
* 변수명[index] (좌)배열 호출 방법
