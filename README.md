## 변수
* 프로그래밍의 첫번째 시작작업은 반드시 필요한 데이터들을 **변수(저장소)**로 저장해야한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 2;`
* 변수선언키워드 : var, let, const
* 생성변수면 - 영문, 숫자조합(숫자는 영문뒤로)
## 변수 작성 시 주의사항
* 변수의 이름은 문자(a~z, A~Z), 숫자(0~9), 달러($), 밑줄(_)이 포함된다.
* 변수의 이름은 숫자로 시작할 수 없다.
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수없다.
* 키워드나 예약어는 사용할 수없다.
    * 키워드 예약어 종류 -> `delete, this, false, ture, if, default, var, new, class`등
* 모든 변수는 대/소문자를 구별합니다. `score`와 `Score`는 다른 변수!
* ★ 변수가 저장할 정보의 종류를 표현하는 의미 있는 단어를 사용해야함!
## 변수 표기법 종류
* 캐멀 표기법 : 두번째 새로운 단어가 나타났을 때, 글자만 대문자로 표기(낙타혹)
    * 예) newName, WebSiteCode
* 파스칼 표기법 : 앞에서 부터 대문자로 표기(개발, 백엔드에서 사용)
    * 예) NewName, WebSiteCode
* 밑줄 표기법 : 단어 사이에 _표기
    * 예) new_name, web_site_code
## 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 객체를 메서드방법으로 실행함
* `객체.속성.메서드();`
* `객체.속성.속성.메서드();`
* `객체.메서드().속성;`
* `객체.메서드().메서드();`
* `객체.메서드(메서드());`
## 데이터타입 구분 함수 `typeof()`
* css 의 배경색상으로 영역구분하는것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1;`
    * `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수 확인
    * `console.log(type(num));` //Number(숫자)
    * `console.log(type(str));` //String(문자)
## null과 undefind 차이
### null
* 값이 비어있음, 존재하지않음
* 사용자에 의해 결정되는 의도적인 대입 값
### undefind
* 정의(할당)되지않음
* undefind는 변수를 초기화하지않은 오류상태
* undefind를 변수에 대입해선 안된다
* 사용자가 실수로 초기화하지않은 변수에 들어간 일종의 오류값
## 산술연산자 -> 피연산자가 숫자인경우
* `a+b` 더하기
* `a-b` 빼기
* `a*b` **★곱하기**
* `a/b` **★나누기**
* `a%b` **★나머지** : 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지이다
* `a**b` 거듭제곱 : 왼쪽의 피연산자의 오른쪽 피연산자의 값만큼 곱한 값이다.
* 계산할 수 없는 경우는 NaN 처리 -> ex)'two'*2=NaN
## 연결연산자(+), 더하기연산자(+)
* `문자열 + 문자열``'hello' + 'world'` -> 문자열 결합으로 더하기 연산이 아닌 연결 연산자로 인식
* `문자열 + 숫자``'hello' + 7` -> 문자열 결합으로 더하기 연산이 아닌 연결 연산자로 인식
* `숫자 + 숫자``1 + 7` -> 숫자 + 숫자 -> 숫자 결합으로 더하기 연산자로 인식
* `(숫자 + 숫자) + 문자열``(1+7) + 'world'` -> 앞 순자는 더하기로 처리후 최종적으로 문자열로 결합
## 빼기(-) 곱하기(*) 나누기(/) 연산자의 특별한 기능
* 더하기 연산자는 피연산자에 따라 문자열이 하나라도 있으면 문자열 결합으로 결과가 나타나지만 빼기, 곱하기, 나누기 연산자는 **숫자형으로 자동 형변환하여 인식**
* `문자열 * 숫자``'1'* 7` -> 문자를 숫자로 자동 형변환하여 결과값을 처리
* `숫자 / 숫자``1 / 7` -> 숫자로 인식 후 결과값을 처리
* `(숫자 + 숫자) - 문자열``(1 + 7) * '2'` -> 문자를 숫자로 자동 형변환하여 결과값을 처리
